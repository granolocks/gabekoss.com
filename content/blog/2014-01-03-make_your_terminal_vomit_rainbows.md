Having a bad day? Why not make your _TERMINAL VOMIT RAINBOWS_? Install the
`lolcat` gem with `gem install lolcat`, then run the following and brace
yourself!

```bash
cat /dev/urandom  | xxd | lolcat -a -d 1 -s 30.0
```
