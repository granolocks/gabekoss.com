cp

cp is a common unix utility to copy a file.

Simple usage: 

cp old_file new_file

Include a timestamp

Copy a file to a new copy including a current string formatted timestamp.

Source of this command.

cp old_file new_file.`date +"%Y%m%d%H%M%S"`
