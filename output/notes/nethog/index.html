nethog

Nethog is a tool similar to top which will show you some useful tabular information on network activity for a given interface including relevant process / PID.

Run with:

sudo nethog <interface>

Output behaves like top (updates regularly) and looks kind of like this: 

NetHogs version 0.8.0

PID   USER         PROGRAM                                                 DEV        SENT      RECEIVED
